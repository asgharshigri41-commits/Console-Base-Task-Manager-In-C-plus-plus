# Console-Based Task Manager in C++

This project is a **Console-Based Task Manager developed in C++** as part of my **first C++ internship project offered by Code Alpha**.
The program allows users to manage their daily tasks efficiently through a simple command-line interface.

## Project Overview

This application helps users organize and manage their daily activities using a console-based environment. It provides basic task management features such as adding tasks, deleting tasks, marking tasks as completed, and saving tasks for future use.

The goal of this project is to practice **fundamental C++ programming concepts**, including Object-Oriented Programming (OOP) and file handling, while building a practical and useful application.

## Features

* Add new tasks with descriptions
* View the list of all tasks
* Mark tasks as completed
* Delete tasks that are no longer needed
* Save tasks to a file for permanent storage
* Load saved tasks when the program runs again

## Concepts Used in Building This Project

### 1. Basic C++ Concepts

* C++ syntax and program structure
* Variables and data types
* Input and output using `cin` and `cout`

### 2. Control Structures and Data Structures

* `vector` to store tasks dynamically
* Loops (`for`, `while`) to repeat operations
* Conditional statements (`if-else`) to control program flow

### 3. Object-Oriented Programming and File Handling

* Classes and objects to represent tasks
* Encapsulation to organize task-related data and functions
* File handling (`fstream`) to save and load tasks for permanent storage

## Purpose of the Project

The purpose of this project is to strengthen my understanding of **C++ programming, problem-solving, and basic software development practices** by building a real-world console application.
